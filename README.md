BJT Amplifiers using ngspice
Overview

This repository contains the design and simulation of the three fundamental BJT amplifier configurations using ngspice:

Common Emitter (CE) Amplifier
Common Collector (CC) Amplifier
Common Base (CB) Amplifier

The project focuses on:

Circuit design using SPICE netlists
Transient waveform analysis
Input and output waveform observation
Understanding amplifier behavior

These circuits were created for analog electronics and VLSI learning practice.
Learning Outcomes

Through this project:

Understood BJT amplifier configurations,
